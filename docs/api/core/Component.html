Namespace: Component

Component

The description object of a UIComponent that is
related to a Blaze Template with the same name.

Properties:
Name Type Description
name string

the name of the Template, must be strictly equal to the Template's name
class string

the (static) base classes of the component
state function

optional state factory
attributes object | function | undefined

define default attributes that always have to exist,
for example role="button" or type="submit". Can be overridden at runtime. If you provide a function you
can precisely control how attributes are resolved and attached to the component.
Leave undefined if no variants and no default attributes are used.
variants object | function | undefined

a dictionary for variants and their respective values.
see the npm package class-variant-authority for what this is and how this works.
defaultVariants object | undefined

a dictionary to defined default values for the given variants
onCrated function | undefined

optional Blaze Template lifecycle method
onRendered function | undefined

optional Blaze Template lifecycle method
onDestroyed function | undefined

optional Blaze Template lifecycle method
helpers object | function | undefined

optional Blaze Template lifecycle method argument
events object | function | undefined

optional Blaze Template lifecycle method argument
