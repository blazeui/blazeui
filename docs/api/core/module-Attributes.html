Type Definitions

AttributesObject

Global attributes resolution handler.
Hooks into a Template instance's lifecycle using
onCreated and onDestroyed to register / release
a reactive object that holds all element attributes.

The target template will be rendered with all necessary attributes
if, and only if props or state changes.
Usually you don't have to manage this on your own.
