Namespace: State

State

Implements shared state between parent and children.
This allows for higher-order-components (hoc) that include
complex logic and where children might or might not
by included in the actual implementation.

The parent or root component defines a context that is
passed down. A global blazeui_context helper is
defined to easily provide it.

Any child component that wants to access or modify the state
needs to link with this state.
For this, children can use the State.useFromContext method.
