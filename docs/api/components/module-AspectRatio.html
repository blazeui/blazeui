Module: AspectRatio

A simple wrapper for images and other visual elements
that enforces the given aspect ratio.

Source:
  • aspect-ratio/AspectRatio.js, line 3
Example
{{#AspectRatio ratio="20/7"}}
  <img src="blazeui_wide.png"
       alt="BlazeUI logo wide"
       class="h-full w-full rounded-md object-contain"/>
{{/AspectRatio}}
